# Pandas & Data Analysis – Week 3 Internship

This repository contains my **Week 3 internship tasks** focused on learning data analysis using the **Pandas library**.  
The goal of this week was to understand how to work with structured data, perform data manipulation, and analyze datasets effectively.

---

## Topics Covered

### Day 1 – Pandas Series & DataFrames
- Understanding Series (1D) and DataFrames (2D)
- Creating data structures from lists and dictionaries
- Exploring data using `.head()`, `.tail()`, `.describe()`
- Checking data types and basic properties

### Day 2 – Import & Export Data
- Reading data using `pd.read_csv()` and `pd.read_excel()`
- Writing data using `.to_csv()` and `.to_excel()`
- Working with parameters like `index_col`, `header`, and `sep`
- Handling small datasets (student data, sales data)

### Day 3 – Indexing & Filtering
- Using `.loc[]` and `.iloc[]` for selection
- Filtering data using conditions
- Boolean indexing and slicing
- Extracting subsets from datasets

### Day 4 – Grouping & Aggregation
- Using `.groupby()` for data grouping
- Applying aggregation functions (`sum`, `mean`, `count`)
- Using `.agg()` for multiple operations
- Analyzing grouped data (e.g., average scores, survival rates)

### Day 5 – Merging & Joining Datasets
- Combining datasets using `pd.merge()`
- Types of joins: inner, outer, left, right
- Concatenating data using `pd.concat()`
- Handling duplicate and overlapping columns

### Day 6 – Practice Session (Titanic Dataset)
- Loading and exploring dataset
- Handling missing values
- Applying filtering, grouping, and aggregation
- Performing analysis such as:
  - Survival rate by gender/class
  - Average age of survivors
